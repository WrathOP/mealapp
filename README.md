# Meals App

[![Demo](https://meals-app-riverpods.web.app/)](https://meals-app-riverpods.web.app/)

## Overview

Meals App is a Flutter-based application that displays various categories of meals. Users can filter meals based on dietary preferences such as gluten-free, vegetarian, and more. This project is deployed on the web using Firebase for demonstration purposes.

## Features

- Display meals in different categories
- Filter meals based on dietary preferences
  - Gluten-free
  - Vegetarian
  - Vegan
  - Lactose-free

## Technologies Used

- Flutter
- Firebase
